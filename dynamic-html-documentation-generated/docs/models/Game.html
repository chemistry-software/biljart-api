Game

  • id : String
    The unique identifier for the game
  • competition_id : String
    The ID of the competition the game belongs to
  • player1_id : String
    The ID of the first player
  • player2_id : String
    The ID of the second player
  • date_played : Date
    The date and time when the game was played
  • ongoing : Boolean
    Indicates if the game is currently ongoing
  • outcome : String
    The outcome of the game
    Enum:
    win
    lose
    draw
    maximum_turns
    ongoing
  • points_player1 : Integer
    Points scored by player 1 in the game
  • points_player2 : Integer
    Points scored by player 2 in the game
  • turns : Integer
    The amount of turns taken
